# Bilingual Translation Website

## 🌍 Assignment 2: Project Documentation and Implementation

A professional bilingual translation website providing seamless translation services between **English and Spanish** using modern web technologies and translation APIs.

### 🚀 **Features**
- **Bidirectional Translation:** English ↔ Spanish with instant language swapping
- **Real-time Character Counting:** 5000 character limit with visual feedback  
- **Responsive Design:** Optimized for desktop, tablet, and mobile devices
- **Modern UI/UX:** Beautiful gradient design with smooth animations
- **Copy to Clipboard:** One-click translation copying functionality
- **Error Handling:** Comprehensive error management with user notifications

### 🔗 **Quick Start**
1. Open `index.html` in any modern web browser
2. Select your desired translation direction (English → Spanish or Spanish → English)
3. Enter text in the source language field
4. Click "Translate" or use Ctrl+Enter for quick translation
5. Copy the translation using the copy button
